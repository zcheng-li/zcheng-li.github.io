# UoG Notes 索引

这里汇总的是仓库根目录 `notes/UoG` 下的课程手写笔记以及考试答案。  
点击每门课链接会直接跳转到对应 GitHub 文件夹。

笔记和答案可能有错或不严谨的地方, 请仔细辨别!

## 三年级课程

- [ENG3014-Communication System](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG3014-CSys)  
  通信系统笔记：信息论、傅里叶、Modem、光纤接入等。

- [ENG3015-Control 3](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG3015-C3)  
  控制课程笔记与历年题整理。

- [ENG3023-Electronic Compatibility](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG3023-EC)  
  电磁场与天线相关笔记。

- [ENG3024-Electronic Circuit Design](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG3024-ECD)  
  电子电路设计：放大器、差分放大、电流镜、反相器等。

- [ENG3041-Power Electronics](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG3041-PE)  
  电力电子课程资料与题库。

- [ENG3043-Real Time Computer System](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG3043-RTCS)  
  实时控制系统：体系结构、进程调度、中断、汇编等。

- [ENG3091-Advanced Programming and Software Engeering](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG3091-APSE)  
  APSE 课程笔记。

## 四年级课程

- [ENG4042-Control 4](https://github.com/zcheng-li/zcheng-li.github.io/tree/main/notes/UoG/ENG4042-C4)  
  高阶控制：可控/可观、极点映射、状态反馈、PID 等。
