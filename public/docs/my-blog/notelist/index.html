    UoG Notes 索引 #

    这里汇总的是仓库根目录 notes/UoG 下的课程手写笔记以及考试答案。
    点击每门课链接会直接跳转到对应 GitHub 文件夹。

    笔记和答案可能有不严谨甚至错误的地方, 请仔细辨别!

    三年级课程 #

    • ENG3014-Communication System
      通信系统笔记：信息论、傅里叶、Modem、光纤接入等。

      其中最后的大作业可以参考 动态 QAM 调制的 python 实现

    • ENG3015-Control 3
      控制课程笔记与历年题整理。

    • ENG3023-Electronic Compatibility
      电磁场与天线相关笔记。

    • ENG3024-Electronic Circuit Design
      电子电路设计：放大器、差分放大、电流镜、反相器等。

    • ENG3041-Power Electronics
      电力电子课程资料与题库。

    • ENG3043-Real Time Computer System
      实时控制系统：体系结构、进程调度、中断、汇编等。

    • ENG3091-Advanced Programming and Software Engeering
      APSE 课程笔记。

    四年级课程 #

    • ENG4042-Control 4
      高阶控制：可控/可观、极点映射、状态反馈、PID 等。

    • ENG4053-Digital Signal Processing 4
      数字信号处理：DTFT/DFT、采样、差分方程、自适应 FIR。

    • ENG4184-Navigation System 4
      导航系统：INS/GNSS、ECEF/NED、惯导方程、传感器模型。

    • ENG4187-Power Electronic and Drives 4
      电机与驱动：PM 电机、感应电机、FOC、实验资料与历年题。
